Creating a fine-grained token with minimal permissions:
  1. Go to github.com → Settings → Developer settings → Fine-grained tokens → Generate new token
  2. Set Token name to something like mermaid-forge
  3. Set Expiration to your preference (e.g. 90 days)
  4. Under Repository access, select "Only select repositories" and pick your diagrams repo
  5. Under Permissions → Repository permissions, set Contents to Read and write. Leave everything else at "No access"
  6. Click Generate token and paste it above
This grants the absolute minimum: read/write files in one specific repo. No access to other repos, issues, PRs, settings, or anything else.
